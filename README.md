# exTimeline

exTimeline is a small, free Extension to help you create timelines in the same sense as GameMaker Studio already provides but with extra features:

* Have more than one timeline per object! Great for complex object behavior!
* Group timelines into classes and manipulate them all at once!
* Functions for Playback, Pause/Resume, Speed for each timeline, classes and all of them!
* Address timelines by their unique name. No global mess!
* Use scripts for your triggers, supports arguments too!
* auto sync your timelines to delta_time if needed
